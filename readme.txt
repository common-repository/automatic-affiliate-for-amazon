This plugin helps you to display Amazon Products on Posts, Pages and your Sidebar. 
Use Shortcodes or let the plugin automatically insert Amazon links.

== Description ==

The Automatic Amazon Affiliate Plugin for Wordpress offers a convenient way to monetize your articles. Each article will be parsed for Amazon product links and added as product box into the sidebar widget. Receive more clicks on your affiliate links through highly visible and relevant Amazon products in your sidebar. And one of the best things - it won't get recognized by an AdBlocker and is fully responsive!

Feature Overview:

- AUTOMATIC PRODUCT LINKS
Save time! The Plugin automatically detects Amazon products for each article and publishes them in a highly visible sidebar widget!

- REALTIME PRICE UPDATES
Never worry about updating prices again! The plugin features automatic price updating through AWS API!

- PRODUCT IMAGES
Profit from higher click-through-rates on your affiliate links with large product images from Amazon is automatically added via AWS API!

- SHORTCODE MODE
As an alternative to the sidebar widget you can employ the shortcode that allows including the box directly into articles wherever you want it!

Plug-In Modes:

- SHORTCODE #1, Amazon Active Link:
[fs14_amazonlink aid="##########"]
Demo: http://www.wordpress-affiliate-plugins.com/amazon-active-link-demo/

- SHORTCODE #2, Amazon Bestseller List:
[fs14_bestseller keyword="Keyboard" limit=3 detail=1]
Demo: http://www.wordpress-affiliate-plugins.com/amazon-bestlist/

- AUTOMATIC Mode
Demo: http://www.wordpress-affiliate-plugins.com/amazon-automatic-sidebar-demo/

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/fs-14-aaap` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->FS-14 AAAP screen to configure the plugin
4. Use the Widget-Manager to add the FS-14 AAAP Widget to your sidebar. (You can deactivate the plugin for each article via article settings)
